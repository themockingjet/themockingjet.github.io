# themockingjet

GitHub Pages site for [themockingjet](https://themockingjet.github.io) portfolio.

## Technologies Used

- Vite
- React
- TypeScript
- Tailwind CSS
- ESLint
- Prettier

## Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/themockingjet/themockingjet.github.io.git
cd themockingjet.github.io
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173` to view the site.

## Build for Production

To build the project for production, run:

```bash
npm run build
```

The built files will be located in the `dist` directory.

## Preview Production Build

To preview the production build locally, run:

```bash
npm run preview
```
